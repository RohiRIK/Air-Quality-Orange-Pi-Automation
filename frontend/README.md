# Frontend Application

This is the Next.js frontend application for the Air Quality Monitor. It displays sensor data and AI-powered recommendations.

## How to Run (Docker)

This application is part of the `docker-compose` setup. To run it, navigate to the project root and execute:

```bash
docker-compose up --build frontend
```

## Key Features

*   Displays real-time sensor data.
*   Shows AI-generated recommendations.
*   Responsive design.
